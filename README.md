## Uuden react-apin luominen 
Luodaan sovellus nimeltään [nimi] ja mennään sovelluksen 
sisältämään hakemistoon:

$ npx create-react-app [nimi]     
$ cd [nimi]    
Sovellus käynnistetään seuraavasti    
$ npm start
